Dodge Car Website 

A modern and responsive Dodge car company website built using React, Vite, Tailwind CSS, and React Router.
The website showcases Dodge vehicles with a clean UI, smooth navigation, and a professional car dealership layout.

🌐 Features

Modern and responsive design

Hero section with Dodge car showcase

Cars listing page with grid layout

Individual car details page

About page describing the company

Contact page with form and location

Sticky navigation bar

Mobile-friendly responsive layout

🛠 Tech Stack

React 18

Vite

Tailwind CSS

React Router

JavaScript (ES6)
